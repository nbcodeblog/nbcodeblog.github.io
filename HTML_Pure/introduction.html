HTML Introduction

What is HTML ?

  • HTML stands for Hyper Text Markup Language.
  • HTML is the standard markup language for creating Web pages.
  • HTML describes the structure of a Web page.
  • HTML consists of a series of elements.

HTML Tags

  • An HTML tag is defined by a start tag, and end tag.
Start Tag End Tag
<h1> </h1>
<p> </p>

HTML Empty Tags

  • An HTML empty tag is defined by a start tag, and no end tag.
Start Tag End Tag
<h1> none
<p> none

HTML Elements

  • An HTML element is defined by a start tag, some content, and an end tag.
Start Tag Element Content End Tag
<h1> Paragraph </h1>
<p> Paragraph </p>

HTML Empty Elements

  • An HTML empty element is defined by a start tag, no content, and an no end tag.
Start Tag Element Content End Tag
<h1> none none
<p> none none

HTML Attributes

  • HTML attributes provide additional information about HTML elements.
  • All HTML elements can have attributes.
  • Attributes provide additional information about elements.
  • Attributes are always specified in the start tag.
  • Attributes usually come in name/value pairs like: name="value".
Start Tag with Attribute Element Content End Tag
<h1 style="color:red;"> Heading </h1>
<p style="color:green;"> Paragraph </p>

HTML Page Structure

<!DOCTYPE html>
<html>
<head>
<title>Title</title>
</head>
<body>
<h1>Heading</h1>
<p>Paragraph</p>
</body>
</html>

Explain HTML Page Structure

  • <!DOCTYPE html> : It defines that this document is an HTML5 document.
  • <html> : It is the root element of an HTML page.
  • <head> : It contains meta information about the HTML page.
  • <title> : It specifies a title for the HTML page.
  • <body> : It defines the document's body, and is a container for all the visible contents.
  • <h1> : It defines a large heading.
  • <p> : It defines a paragraph.

Web Browsers

  • The purpose of a web browser is to read HTML documents and display them correctly.
  • HTML elements tell the browser how to display the content.
  • The content inside the body section will be displayed in a browser.
  • A browser does not display the HTML tags, but uses them to determine how to display the document.

HTML Documents

  • All HTML documents must start with a document type declaration.
  • The HTML document itself begins with html and ends with html.
  • The visible part of the HTML document is between body.

The <!DOCTYPE> Declaration

  • The <!DOCTYPE> declaration represents the document type, and helps browsers to display web pages correctly.
  • It must only appear once, at the top of the page (before any HTML tags).
  • The <!DOCTYPE> declaration is not case sensitive.
  • <!DOCTYPE> This is HTML5 doctype declaration.

HTML Source Code

View HTML Source Code

  • Click CTRL + U in an HTML page or
  • right-click on the page and select "View Page Source"
  • This will open a new tab containing the HTML source code of the page.

Inspect an HTML Element

  • Right-click on an element (or a blank area), and choose "Inspect" to see what elements are made up of.
  • You can also edit the HTML or CSS on-the-fly in the Elements or Styles panel that opens.

Nested HTML Elements

  • HTML elements can be nested (this means that elements can contain other elements).
  • All HTML documents consist of nested HTML elements.

HTML is Not Case Sensitive

  • HTML tags are not case sensitive: <P> means the same as <p>.
  • The HTML standard does not require lowercase tags, but W3C recommends lowercase in HTML, and demands lowercase for stricter document types like XHTML.
